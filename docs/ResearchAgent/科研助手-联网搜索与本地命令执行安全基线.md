# 科研助手联网搜索与本地命令执行安全基线（P0）

## 1. 文档目标与适用范围
- 目标：为 `P0` 阶段“联网搜索 + 本地命令执行”能力提供可直接落地的安全控制基线，确保可运行、可审计、可管控。
- 适用范围：用户侧与智能体侧执行链路中的检索工具、本地命令工具、人工干预流程、审计事件流。
- 非目标：不包含管理端页面实现；管理端仅保留审计查询与强停等后端兼容字段。

## 2. 安全基线原则
- 默认拒绝：未在允许列表中的来源、命令、参数一律拒绝。
- 最小权限：工具仅获得完成当前任务所需的最小网络与本地执行权限。
- 显式确认：高风险动作必须进入人工确认，未确认不得执行。
- 全程审计：每次工具调用都必须生成可追溯事件，绑定 `task_id` 与 `step_id`。
- 失败可解释：被拦截、降级、超时、重试均需输出可解释原因。

## 3. 风险分层与控制总览

### 3.1 联网搜索风险分层
- `L1 低风险`：已备案学术站点检索（只读、公开内容）。
- `L2 中风险`：未知站点但通过协议/格式校验，需沙箱抓取与严格超时。
- `L3 高风险`：非白名单域名、可疑重定向、下载型链接、脚本注入特征，直接阻断或转人工确认。

### 3.2 本地命令风险分层
- `C1 低风险`：纯只读命令模板（目录列举、文件摘要、哈希计算）。
- `C2 中风险`：受限写入命令模板（仅允许临时工作目录内生成中间产物）。
- `C3 高风险`：系统级变更、网络外连、权限提升、跨目录写入、可执行脚本注入，默认阻断并要求人工确认（通常不放行）。

## 4. 联网搜索安全基线

### 4.1 域名与来源白名单
- 允许域名采用显式白名单管理，按来源类型分组：
  - 学术来源：如期刊、预印本、学术搜索服务。
  - 公共知识来源：官方文档与公共机构知识库。
- 白名单校验包含：`scheme`、主域名、端口、路径规则。
- 禁止项：
  - 非 `https`（特殊受控内网场景除外）；
  - 通过短链或多级重定向绕过域名校验；
  - 下载可执行文件、压缩包、二进制脚本。

### 4.2 请求治理与反滥用
- 单任务并发阈值：限制同任务外部请求并发数，避免突发流量。
- 速率治理：节流 + 随机抖动 + 指数退避重试，降低触发站点风控概率。
- 超时配置：
  - 连接超时：短超时（如 3~5s）；
  - 读取超时：中超时（如 10~20s）；
  - 单步骤总预算超时：统一上限（超限后进入降级）。
- 输出截断：正文、摘要、元数据长度受限，避免污染上下文和过量存储。

### 4.3 结果质量与可信度约束
- 每条检索结果统一结构化字段：
  - `query`、`source`、`url`、`published_at`、`confidence`、`snippet`。
- 禁止“无来源结论”进入最终报告；引用必须可回溯到来源 URL 与步骤。
- 当检索命中不足或可信度不足时，执行“再检索”或回退到本地 RAG。
- 对异常站点返回结果要标记 `risk_flag`，默认不进入高置信证据池。

### 4.4 联网异常处置策略
- 域名不在白名单：立即拒绝并记录 `blocked_domain` 事件。
- 站点限流/验证码：自动降速并切换替代来源；连续失败后给出可解释失败。
- 内容不可解析：降级为元信息模式，仅保留标题、时间、来源。
- 可疑内容注入：移除脚本、可执行片段、外链重定向参数，必要时丢弃结果。

## 5. 本地命令执行安全基线

### 5.1 命令模板白名单
- 仅允许模板化命令调用，不允许自由 shell 拼接。
- 每个模板必须定义：
  - `template_id`、用途说明、参数 schema、允许目录、超时、输出上限、风险级别。
- 参数必须通过类型与取值校验（长度、字符集、枚举值、路径归一化）。
- 明确禁止：
  - 命令串联（`&&`、`|`、`;`、重定向注入）；
  - 动态拼接可执行路径；
  - 任意网络访问命令；
  - 权限提升命令与系统服务管理命令。

### 5.2 执行环境隔离
- 工作目录隔离：仅允许在任务级临时目录执行，禁止跨目录写入。
- 资源限制：
  - CPU/内存/进程数限制；
  - 单命令超时强杀；
  - 标准输出/错误输出截断。
- 环境变量最小化：禁止继承敏感凭证与主机关键环境变量。
- 文件访问策略：默认只读，写入需模板显式声明并限定路径。

### 5.3 人工确认与高风险拦截
- 命中高风险规则时，任务进入 `pending_action`：
  - 返回“动作卡片”：动作摘要、风险原因、影响范围、可选操作（允许/终止/修订）。
- 未确认超时策略：
  - 到达等待上限后自动终止该动作并记录 `approval_timeout`。
- 用户修订后需重新进行参数校验与风险评估，不可直接沿用原请求。

### 5.4 本地执行异常处置
- 模板不存在或参数不合规：立即拒绝并写 `command_rejected` 事件。
- 运行超时：强制终止进程，写 `command_timeout`，可按策略重试一次。
- 输出超限：截断输出并标记 `output_truncated=true`。
- 运行失败：返回标准化错误码与可读错误摘要，不暴露主机敏感信息。

## 6. 审计字段标准（统一事件模型）

### 6.1 事件必填字段
- 标识字段：`event_id`、`task_id`、`step_id`、`trace_id`、`timestamp`。
- 主体字段：`user_id`、`actor_type`（user/system/admin）。
- 动作字段：`tool_type`（web_search/local_command）、`action`、`risk_level`。
- 请求字段：`input_snapshot`（脱敏后）、`policy_version`、`rule_hit`。
- 结果字段：`status`（allowed/rejected/failed/succeeded/pending_action）、`error_code`、`error_message`。
- 成本字段：`latency_ms`、`token_used`、`resource_cost`（可选）。

### 6.2 联网搜索扩展字段
- `domain`、`url`、`source_type`、`http_status`、`redirect_chain`。
- `confidence`、`published_at`、`citation_id`、`blocked_reason`（命中拦截时）。

### 6.3 本地命令扩展字段
- `template_id`、`cwd`、`args_snapshot`、`exit_code`、`timeout_ms`。
- `stdout_size`、`stderr_size`、`output_truncated`、`approval_required`、`approval_result`。

### 6.4 图像生成联动字段（按规划预留）
- 当触发 `Qwen_Image` 时，事件需附加：
  - `image_enabled_reason`、`linked_report_section`、`citation_refs`。
- 图像产物必须与同任务文本结论同链路审计，不得脱离 `task_id`。

## 7. 熔断、降级与封禁策略
- 域名熔断：同域名在窗口期连续失败超过阈值，临时熔断并切换来源。
- 成本熔断：任务累计 token 或调用次数超预算，停止非关键步骤。
- 风险封禁：命中高危规则可触发用户级或任务级临时封禁。
- 恢复策略：熔断/封禁均需记录生效与解除事件，支持审计回放。

## 8. 验收检查清单（P0）
- 白名单有效：非白名单域名请求被拦截并返回可解释错误。
- 模板有效：非白名单命令无法执行，参数注入尝试被阻断。
- 人工确认有效：高风险动作进入 `pending_action`，未经确认不执行。
- 隔离有效：命令执行无法写入隔离目录外路径，超时可强制终止。
- 审计完整：任意工具调用都可通过 `task_id + step_id` 追踪全链路事件。
- 降级有效：外部站点失败时能够切换替代源或给出明确失败原因。
